교육 대상

  • 경력단절 이후 재취업/재도전을 준비하는 분
  • 포토샵이 처음이거나 기초부터 다시 배우고 싶은 분
  • 자기소개서·면접·포트폴리오를 체계적으로 만들고 싶은 분
  • AI(생성형 AI) 활용으로 문서 완성도를 높이고 싶은 분

교육 방식 실습·피드백

  • 매 차시: 미니 실습 → 결과물 제출 → 피드백
  • AI는 “대충 쓰는 법”이 아니라 직무/경험 기반 프롬프트로 훈련
  • 최종: 포트폴리오 PDF + 자기소개서 세트 + 면접 답변 스크립트 완성

상세 커리큘럼

A. 포토샵 실무 (26h)

  • 기초 다지기: 화면 구성, 레이어/그룹, 선택/변형, 단축키
  • 이미지 보정: 밝기/색감, 리터칭 기초, 배경 제거
  • 디자인 기본: 타이포, 여백, 컬러, 그리드
  • 템플릿 제작: 카드뉴스/배너/썸네일 템플릿 만들기
  • 실전 제작: 블로그·SNS 콘텐츠 제작 & 개인 브랜드 이미지 통일

B. AI 기반 취업준비 (39h)

  • 직무 탐색: 목표 직무/기업 분석, 요구역량 체크리스트
  • 경험 정리: STAR 방식으로 경험/성과를 문장화
  • 자소서 작성: AI로 초안 생성 → 나의 경험으로 사실 검증 → 문장 다듬기
  • 포트폴리오: 구성(소개/역량/프로젝트/성과) + 시각 템플릿 적용
  • 면접 대비: 예상 질문 생성, 답변 구조화, 모의면접(피드백)
  • 실전 패키지: 제출용 파일 정리(PDF/문서명 규칙/버전 관리)

포인트: AI는 “대충 써서 복붙”이 아니라, 내 경험·직무·기업에 맞게 질문을 설계하고 결과를 검증하는 방식으로 진행합니다.

수료 후 기대 결과

  • 포토샵으로 SNS/블로그/홍보물을 직접 제작·관리할 수 있음
  • 자기소개서를 직무별로 2~3버전 완성 + 문장 품질 향상
  • 포트폴리오 PDF 완성(템플릿 적용, 가독성/구성 강화)
  • 면접 답변 스크립트 및 예상질문 세트로 실전 대비
